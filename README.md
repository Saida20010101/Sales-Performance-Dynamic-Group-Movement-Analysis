Sales Performance & Customer Segment Movement Analysis
 Project Overview:
This project involves the creation of a dynamic Power BI report to track sales performance and customer group transitions. The main objective was to segment customer groups based on their monthly activity and visualize their movements over time.
 Technical Implementations
Advanced DAX Logic: Developed a complex `Group Status Count` measure to dynamically classify groups into Shark, Mid, Beginner, New, Inactive, and Reactivated segments.
Organization:Implemented a Measures Table for scalability and better maintainability of the project.
Key Business Insights
23M Total Sales overview with interactive drill-downs.
 Identification of growth trends via  New and Reactivated group tracking.
 Brand-specific performance analysis to pinpoint high-value group distributions.
